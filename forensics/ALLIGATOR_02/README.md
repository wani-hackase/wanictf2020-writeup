# ALLIGATOR_02

## 問題文

コマンドプロンプトの実行履歴からFLAGを見つけてください。

(ALLIGATOR_01で配布されているファイルを使ってください)


## FLAG
FLAG{y0u_4re_c0n50les_master}

## 解法

### ALLIGATOR_02

cmdscanでは入力したコマンドを、consolesではコマンドに対する出力まで含めて確認することができる。
